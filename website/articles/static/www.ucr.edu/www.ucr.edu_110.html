Getting the information you want and need over the web will be easy with UCR's web portals.

The portals will allow you to build a personalized web page with the content you want, as well as allowing the university to send you important messages and reminders.

The full portal systems — for staff, faculty and students — are expected to be in place in the fall. In the meantime, we have set up this page to begin to familiarize you with using a portal.
